TK01
Название:
Проверка авторизации пользователя (ввод валидных данных)

Шаги:
1. Перейти  на сайт Alefowl.com
2. Нажать кнопку "Login"
3. Заполнить поля "Login" и "Password" валидными данными
4. Нажать кнопку "Log in"

Ожидаемый результат:
Пользователь авторизован на сайте Alefowl.com и попадает
на главную страницу с возможностью зайти в личный кабинет



TK02
Название:
Проверка авторизации пользователя (неверный логин)

Шаги:
1. Перейти  на сайт Alefowl.com
2. Нажать кнопку "Login"
3. Заполнить поля "Login" не валидные данные и "Password" валидные данные
4. Нажать кнопку "Log in"

Ожидаемый результат:
Пользователь не авторизован на сайте, появляется сообщение об ошибке
